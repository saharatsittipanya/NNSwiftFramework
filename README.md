## Requirements

### • Swift 4.2

### • XCode 10.0

### • iOS 12.0

## Installation

NNSwiftFramework is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'NNSwiftFramework'
```

Then save Podfile run this on command line:

```ruby
pod install
```
or
```
pod update
```
